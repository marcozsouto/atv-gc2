
# Atividade GC2

API de films utilizando o github flow


## Documentação da API

#### Requisitos

```http
  "node": "^20.11.21"
```

#### Retorna todos os itens

```http
  GET http://localhost:3001/films
```



## Instalação

Instale utilizando os seguintes comandos

```bash
  npm install 
  npm run build
```

## Execute

Execute

```bash
  npm start 
```
